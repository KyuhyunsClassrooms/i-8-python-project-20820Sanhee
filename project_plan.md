# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제:

 나만의 스마트 공부 습관 점수 분석기


---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

매일 공부는 하지만 내가 어떤 과목을 효율적으로 공부하고 있는지, 계획 대비 얼마나 달성하고 있는지 객관적으로 파악하기 어렵다. 사용자가 공부한 과목과 시간, 집중도를 입력하면 공부 습관을 정량적으로 분석하고 맞춤형 피드백을 주는 프로그램을 만들고자 한다.

---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?
오늘 공부한 총 과목 수 (정수)

과목별 정보 (과목명, 계획했던 시간(분), 실제 공부한 시간(분), 집중도 점수(1~5))
- 
- 
- 

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 과목별 달성률 계산: (실제 시간 / 계획 시간) * 100 (단, 계획 시간이 0일 때의 예외 처리)

학습 효율 점수 산출: (달성률 * 0.6) + (집중도 점수 * 8) (100점 만점 기준의 자체 공식)

최고 효율 과목 탐색: 효율 점수가 가장 높은 과목과 점수를 찾아내는 최댓값 알고리즘 적용
- 
- 

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 과목별 분석 결과 (과목명, 달성률, 효율 점수)

오늘 하루 전체 평균 효율 점수 및 가장 효율이 좋았던 최고 효율 과목

전체 평균 점수대에 따른 '내일의 한 줄 조언' (피드백)
-

## 4. 2차원 리스트 설계

내 프로그램에서 사용할 2차원 리스트의 이름:

# 나만의 스마트 공부 습관 점수 분석기 데이터 리스트
# 형식: study_logs = [ ["과목 이름", 계획 시간, 실제 시간, 집중도 점수], ... ]

study_logs = [
    ["수학", 60, 50, 4],  # 0번 행: 수학 (계획 60분, 실제 50분, 집중도 4)
    ["영어", 40, 45, 5],  # 1번 행: 영어 (계획 40분, 실제 45분, 집중도 5)
    ["정보", 30, 30, 3],  # 2번 행: 정보 (계획 30분, 실제 30분, 집중도 3)
    ["과학", 50, 20, 2]   # 3번 행: 과학 (계획 50분, 실제 20분, 집중도 2)
]
```

### 한 행의 의미
오늘 특정과목을 공부한 기록

### 각 열의 의미

| 열 번호 | 의미 | 예시 |
|---:|---|---|
| 0번 열 |  과목 이름 (문자열) | "수학" |
| 1번 열 |계획했던 시간  |60  |
| 2번 열 |   실제 공부한 시간| 50|
| 3번 열 | 내가 느낀 집중도 점수 | 4|

---

## 5. 함수 설계

| 함수 이름 | 역할 | 입력값 | 반환값 또는 출력 |
|---|---|---|---|
| input_log() | 사용자에게 과목별 정보를 입력받아 2차원 리스트를 생성함 | 없음 | 생성된 2차원 리스트 (study_logs) |
|analyze_efficiency()  | 2차원 리스트를 바탕으로 달성률, 효율 점수, 최고 과목을 계산함 | 2차원 리스트 (logs) | 전체 평균 점수, 최고 효율 과목명 |
|show_report()  | 분석된 결과와 점수별 내일의 조언을 화면에 깔끔하게 출력함 | 평균 점수, 최고 과목명 | 없음 (화면 출력) |


---

## 6. 사용할 제어구조

### 조건문을 사용할 부분


내 계획:최댓값 알고리즘을 통해 각 과목의 효율 점수를 비교하여 가장 높은 과목을 결정할 때

전체 평균 효율 점수가 높은지 낮은지에 따라 내일의 조언(피드백) 문구를 다르게 출력할 때


### 반복문을 사용할 부분

내 계획:사용자가 입력한 과목 수만큼 input()을 반복하며 데이터를 수집할 때

2차원 리스트 study_logs에 저장된 행(과목)을 하나씩 꺼내어 순서대로 달성률과 점수를 계산할 때


---

## 7. 예상되는 예외 상황


1. 사용자가 계획했던 시간에 실수로 0을 입력하여, 달성률 계산 시 0으로 나누기 오류(ZeroDivisionError)가 발생하는 상황
2. 집중도 점수에 1~5 범위를 벗어난 숫자(예: 10점 또는 -5점)를 입력하는 상황

---

## 8. Gemini에게 도움받고 싶은 부분

1. 2차원 리스트를 함수 간에 주고받는 매개변수와 반환값(return) 구조 구현
2. 계획 시간이 0일 때 프로그램이 멈추지 않도록 하는 조건문 예외 처리 방법
3. 계산된 데이터들을 바탕으로 최고 점수 과목을 찾아내는 알고리즘 코드 힌트
